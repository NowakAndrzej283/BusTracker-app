# BusTracker-app 🚎🚌🚃🚈

A mobile application built with React Native that allows users to check the best tranport option depending on their starting point.

## About the App

BusTracker is a mobile App designed to help users find the best Poznań public service vehicle depending on their current location.
It displays bus schedule and provides a useful information navigating within the city of Poznań.

## Features (In Progress)
- Public service vehicles schedule
- Clean and intuitive UI
- Search for bus locations
- Interactive map navigation

## Tech Stack
- React Native
- Expo
- JavaScript
- Maps (React Native Maps)
- External API ( https://www.ztm.poznan.pl/otwarte-dane/gtfsfiles/ )


## Data Source 

The Application uses public transport data provided by the city of Poznań for free (link below).

- GTFS (https://www.ztm.poznan.pl/pl/dla-deweloperow/getGTFSFile)
